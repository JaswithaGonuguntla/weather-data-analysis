# weather-data-analysis
# 🌦️ Weather Data Analysis Dashboard

This project analyzes weather data using Python and Streamlit.

## 🚀 Features
- Upload weather dataset
- Temperature, rainfall, humidity analysis
- Interactive dashboard with filters
- Graphical visualization

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib

## ▶️ How to Run
```bash
pip install streamlit pandas matplotlib
streamlit run app.py
